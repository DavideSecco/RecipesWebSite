# RecipesWebSite

Requisiti:

- python3
- pip3
- pipenv

### Installazione

`git clone https://github.com/DavideSecco/RecipesWebSite](https://github.com/DavideSecco/RecipesWebSite`

`cd RecipesWebSite`

`pipenv shell`

`cd recipes`

`pipenv install django`

`pip install django-cleanup`

`python -m pip install Pillow`

`python manage.py runserver`
